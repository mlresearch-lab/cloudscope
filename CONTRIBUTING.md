# Contributing to CloudScope

## Development Setup

```bash
git clone https://github.com/mlresearch-lab/cloudscope.git
cd cloudscope
pip install -e ".[all]"
pip install pytest
```

## Running Tests

```bash
pytest tests/
```

## Code Style

- Black formatting
- Type hints required
- Docstrings for public APIs

## Pull Requests

1. Fork the repo
2. Create feature branch
3. Add tests for new functionality
4. Ensure all tests pass
5. Submit PR with clear description
